使用列表的一部分
1.切片
创建切片，指定要使用的第一个元素和最后一个元素的索引；在到达指定的第二个索引前面的元素后停止。

如果没有指定第一个索引，python从列表开头开始提取；
同样地，如果没有指定第二个索引，python将一直提取到列表末尾；

3.复制列表
复制列表，可创建一个包含整个列表的切片，方法是同时省略起始索引和终止索引([:])；

如果不使用[:]对列表进行复制的话，实质上food与food1指向同一个列表，也就是说对其中任何一个操作，另一个也随之改变，因为两者是同一个列表。

元组
元组就是不可变的列表。

定义元组

定义元组与列表的区别：
1.使用圆括号()而不是方括号[]来标识
2.元素不带引号

如果尝试修改元组的元素大小，python会报错。

元组的遍历
元组的遍历与普通列表遍历无异。使用for循环即可实现。

修改元组变量
元组的变量是不能直接单个进行修改的。元组是固定不变的。要想修改元组的元素，只能等效的去修改元组变量。

注意：这种修改元组的方式是声明新元组进行覆盖，python允许这样做，因此不会报错。
